A blazingly fast CLI tool to find usernames across hundreds of social media platforms and other websites.

Scan the internet in just 30 seconds using Go's powerful concurrency.
Key Features:
  • Fast scanning
  • Powered by Go's advanced concurrency features
  • Simple and easy to use command-line interface
  • Generates a list of all websites where the username is found
  • Lightweight

Installation

git clone https://github.com/caetano-dev/Gopher-Find
cd gopher-find
go mod tidy
go run cmd/main.go <username>
